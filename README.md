# オーラ鑑定パーフェクトガイド

占い初心者向け「オーラを視る方法」単独教材サイトです。

## 構成
- `login.html` … ログイン画面（パスワード認証）
- `index.html` … サイト本体（解説／ワーク／オーラ一覧表の3タブ構成）
- `images/finger-black-white-bg.png` … 「①背景色は黒か白」セクションで使用する参考画像
- `images/aura-color-master-table.jpg` … オーラ一覧表の元資料画像（14色のプラス・マイナス一覧）
- `netlify.toml` … Netlifyデプロイ設定

**注意：`index.html`単体では画像が表示されません。必ず`images`フォルダも一緒にアップロードしてください。**

## ログイン画面について
パスワードは `oraperfect4932` です。`index.html`に直接アクセスした場合も、未ログインであれば自動的に`login.html`に転送されます。ログイン状態はブラウザのタブを閉じるとリセットされます（セッションストレージ使用のため、簡易的な入室制限です。厳重なアクセス制御が必要な場合は、Netlifyの「パスワード保護」機能の利用もご検討ください）。

## 表示速度について
サイト内で使用している画像枚数が多いため、以下の対応でページの軽さを保っています。
- 同梱画像（一覧表・指先の写真）は表示に適したサイズへ圧縮済み（合計464KB→232KB程度）
- すべての画像に遅延読み込み（loading="lazy"）を設定し、スクロールして表示される直前に読み込む仕組みにしています
- 画像の縦横比をあらかじめ指定し、読み込み中にレイアウトがガタつかないようにしています

## オーラ一覧表について
深紅色・赤色・オレンジ・オレンジイエロー・黄色・緑色・深緑色・青色・藍色・紫色・ラベンダー・白色・黒色・黄金色の14色について、プラス面・マイナス面をまとめたマスターテーブルを掲載しています。印刷ボタンからこの表だけを印刷できます。

## ワークタブについて
選択式・入力式のどちらでも、入力内容に応じて「長所／短所／顕在意識／中層意識／潜在意識／アドバイス」を含むオーラ性格鑑定結果を自動表示します。**この鑑定内容は、すべて「オーラ一覧表」タブの14色マスターデータに基づいています。**あわせて、ChatGPTにそのまま貼り付けられる鑑定プロンプトも自動生成され、「プロンプトをコピー」→「ChatGPTを開く」の2ステップでより詳しい鑑定文を作成できます。

## Netlifyへのアップロード方法（ドラッグ＆ドロップ）
1. https://app.netlify.com/ にログイン
2. 「Add new site」→「Deploy manually」を選択
3. このフォルダ（`aura-guide-deploy`）ごとブラウザにドラッグ＆ドロップ
4. 数秒でデプロイが完了し、公開URLが発行されます
